# Fake Tweet Generator

A fast and interactive web application that allows users to create highly realistic, fully customizable mock X (Twitter) posts and instantly download them as high-quality images. Built with React.js and Tailwind CSS for a pixel-perfect, responsive UI.

**Quick Start:**

```bash
git clone [https://github.com/enesece/fake-tweet-generator.git](https://github.com/enesece/fake-tweet-generator.git)
cd fake-tweet-generator
npm install
npm run dev
```
